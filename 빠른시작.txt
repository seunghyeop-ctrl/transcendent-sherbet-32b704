[빠른 시작]

1. 이 폴더를 팀에서 운영할 서버나 항상 켜둘 컴퓨터에 둡니다.
2. 터미널에서 이 폴더로 이동한 뒤 가상환경을 만들고 `pip install -r requirements.txt` 를 실행합니다.
3. Google AI Studio의 Gemini API Key, Google Sheet ID, credentials.json 을 준비합니다.
4. `python3 app.py --host 0.0.0.0 --port 5001` 로 실행합니다.
5. 같은 네트워크에서는 `http://내부IP:5001` 로 접속합니다.

[포함 파일]
- `app.py`, `web_app.py`, `extract.py`: 최신 웹앱 코드
- `DEPLOY.md`: 배포/운영 가이드
- `deploy/start.sh`: 서버 실행 스크립트

[중요]
- 결과 저장의 메인은 Google Sheets 입니다.
- 영상과 프레임은 임시 처리 후 삭제됩니다.
- Gemini API Key는 웹 설정에서 저장하면 macOS Keychain 에 들어갑니다.
