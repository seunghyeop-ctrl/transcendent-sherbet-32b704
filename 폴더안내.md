# 폴더 안내

## 실사용 위치

- 앱 아이콘: `/Applications/프롬프트 추출기.app`
- 실행 코드: `/Applications/프롬프트 추출기 지원`
- 설정 폴더: `~/Library/Application Support/PromptExtractor`
- 결과 폴더: `~/Documents/프롬프트 추출기`

## 설정 파일

- `config.json`
- `credentials.json`

## 결과 폴더 구조

- `~/Documents/프롬프트 추출기/frames`
- `~/Documents/프롬프트 추출기/outputs`

## 프로젝트 폴더 의미

- 현재 프로젝트 폴더는 소스 수정용
- 앱 실행은 프로젝트 폴더를 참조하지 않음
