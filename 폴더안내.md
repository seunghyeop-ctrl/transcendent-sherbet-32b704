# 폴더 안내

## 현재 프로젝트 의미

- 이 폴더는 `Grommy` 웹앱의 **소스 코드 저장소**입니다.
- 다른 맥북이나 윈도우 PC에서 작업을 이어가려면 이 폴더 전체를 그대로 복사하거나, Git 원격 저장소를 통해 동기화하면 됩니다.

## 코드와 로컬 데이터는 분리됩니다

이 저장소에는 **코드와 문서만** 두는 것이 원칙입니다.

Git으로 공유하지 않는 항목:
- API 키
- `credentials.json`
- 로컬 결과 CSV/XLSX
- 캐시, 로그, 임시 산출물

위 항목들은 `.gitignore`로 제외되어 있습니다.

## 설정 폴더

설정은 운영체제별 사용자 데이터 폴더에 저장됩니다.

- macOS: `~/Library/Application Support/PromptExtractor`
- Windows: `%APPDATA%\\PromptExtractor`
- Linux: `~/.config/PromptExtractor`

주요 파일:
- `config.json`
- `credentials.json`
- `secrets.json` (운영체제/환경에 따라 사용)

## 결과 폴더

기본 결과 폴더는 사용자 문서 폴더 아래에 생성됩니다.

- 기본값: `~/Documents/프롬프트 추출기`

그 아래 자동 생성:
- `outputs/results.csv`
- `outputs/results.xlsx`
- `outputs/results_meta.json`

## 동기화 관련 핵심

- **이 프로젝트는 이제 Git 저장소입니다.**
- 하지만 **자동 최신화는 아직 아닙니다.**
- 집 맥북, 회사 맥북, 윈도우 PC가 서로 자동으로 맞춰지려면
  - GitHub 같은 **원격 저장소**를 만들고
  - 각 기기에서 `push / pull` 하도록 연결해야 합니다.
