# 프롬프트 추출기 사용법

## 실행

1. `응용 프로그램`에서 `프롬프트 추출기.app` 실행
2. `실행` 탭에 링크를 여러 줄로 입력
3. `실행` 버튼 클릭

## 설정

설정은 자동으로 아래에 저장됩니다.

- `~/Library/Application Support/PromptExtractor/config.json`
- `~/Library/Application Support/PromptExtractor/credentials.json`

앱의 `설정` 탭에서 바꿀 수 있는 항목:

- Google Sheet ID
- 워크시트 이름
- credentials.json 경로
- 출력 폴더

## 결과 저장 위치

기본 출력 폴더:

- `~/Documents/프롬프트 추출기`

그 아래 자동 생성:

- `frames/`
- `outputs/results.csv`
- `outputs/results.xlsx`

## Google Sheets

- URL 기준으로 기존 행은 업데이트
- 새 URL은 새 행으로 추가
- 저장 컬럼은 `URL`, `IMAGE`, `PROMPT`, `KO`, `CAMERA`

## 프로젝트 폴더와의 관계

- 앱은 프로젝트 폴더 없이도 동작
- 프로젝트 폴더는 소스 보관용
