# Grommy 사용법

## 실행

프로젝트 폴더에서 실행:

```bash
python3 app.py --host 0.0.0.0 --port 5007
```

접속:
- 내 컴퓨터: `http://127.0.0.1:5007`
- 같은 네트워크 기기: `http://내부IP:5007`

## 설정

웹의 `설정` 화면에서 아래 항목을 관리합니다.

- Gemini API Key
- Google Sheet ID
- 워크시트 이름

설정 파일은 운영체제별 사용자 데이터 폴더에 저장됩니다.

- macOS: `~/Library/Application Support/PromptExtractor`
- Windows: `%APPDATA%\\PromptExtractor`
- Linux: `~/.config/PromptExtractor`

## 결과 저장

기본 결과 폴더:

- `~/Documents/프롬프트 추출기`

주요 결과 파일:
- `outputs/results.csv`
- `outputs/results.xlsx`
- `outputs/results_meta.json`

영상과 프레임은 메인 추출 경로에서 **임시 파일로만 처리**되고, 작업 후 자동 삭제됩니다.

## Google Sheets

- 추출 결과는 Google Sheets와 동기화할 수 있습니다.
- URL 기준으로 기존 행 업데이트 / 새 행 추가가 이뤄집니다.
- 생성 초안도 시트에 저장되며, 삭제 시 시트에서도 함께 삭제됩니다.

## 다른 컴퓨터에서 이어서 작업하기

소스 코드만 옮기면 되는 것은 아닙니다.

새 컴퓨터에서 필요한 것:
1. 이 저장소 코드 받기
2. Python 환경 설치
3. `pip install -r requirements.txt`
4. Gemini API Key 다시 설정
5. `credentials.json` 다시 배치
6. 필요하면 Google Sheet ID 다시 설정

## 자동 최신화 여부

- 현재는 **로컬 Git 저장소**만 만들어진 상태입니다.
- 그래서 집 맥북에서 수정한 내용이 회사 맥북에 **자동으로 반영되지는 않습니다.**
- 자동처럼 쓰려면 GitHub 같은 **원격 저장소**를 연결하고,
  각 컴퓨터에서 `push / pull` 해야 합니다.
