# 팀 배포용 웹앱 가이드

## 용도
- 인스타그램 릴 링크를 넣으면 영상을 임시 다운로드한 뒤 Gemini가 직접 분석합니다.
- 결과는 `카메라 워킹 / 프롬프트(영문) / 내용 / 링크` 형식으로 저장됩니다.
- 메인 저장소는 Google Sheets 입니다.

## 기본 실행
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py --host 0.0.0.0 --port 5001
```

## 필요한 설정
- Gemini API Key
- Google Sheet ID
- 서비스 계정 `credentials.json`

## 접속
- 같은 와이파이: `http://내부IP:5001`
- 외부 공개 운영은 별도 서버 또는 터널이 필요합니다.

## 현재 반영 사항
- direct-video Gemini 분석 메인 경로
- 최근 결과 검색 / 카메라 워킹 태그 필터
- 프롬프트 더 보기
- 항목별 삭제 / 전체 삭제
